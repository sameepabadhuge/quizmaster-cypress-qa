 QuizMaster Cypress Automation

 Project Overview
This project contains automated end-to-end (E2E) tests for the QuizMaster application using Cypress.

Features Tested
- Student Registration
- Student Login
- Full User Flow (Register → Login → Dashboard)
- Negative Login Test

Tools Used
- Cypress
- JavaScript
- Node.js

Test Scenarios
1. Register new student with dynamic data
2. Login with valid credentials
3. Validate dashboard navigation
4. Handle invalid login attempts

How to Run

```bash
npm install
npx cypress open
